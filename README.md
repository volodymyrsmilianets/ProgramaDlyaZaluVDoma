Мета проєкту

Створити мінімально життєздатний продукт (MVP) веб-додатку для формування персональних тренувальних програм.
Основна задача — дозволити користувачу обрати тренувальну мету, рівень підготовки та отримати індивідуальний план тренувань у зручному інтерфейсі.

Використані технології

HTML5 — розмітка сторінок
CSS3 — стилі та адаптивний дизайн
JavaScript — логіка роботи додатку
LocalStorage — збереження даних користувача
Git / GitHub — контроль версій та хостинг проєкту

Функціональність (MVP)

Вибір тренувальної цілі (схуднення, набір маси, тонус тощо)
Вибір рівня підготовки (початківець, середній, просунутий)
Генерація персональної тренувальної програми
Збереження тренування у LocalStorage
Можливість перегляду збережених програм
