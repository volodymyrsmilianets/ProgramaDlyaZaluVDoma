# TRpoint | Персональний Генератор Тренувальних Програм

## Огляд Проєкту

TRpoint — це веб-додаток, призначений для генерації індивідуальних тренувальних програм. Система автоматично створює план тренувань, адаптований під обрану користувачем фітнес-ціль (схуднення, м'язова маса, витривалість) та його рівень підготовки (новачок, середній, просунутий).

Ключовою особливістю проєкту є надійна система авторизації та збереження даних, яка прив'язує профіль, статистику та збережені програми до унікального облікового запису користувача, забезпечуючи їхнє збереження навіть при зміні пристроїв або вході іншого користувача.

Проєкт реалізовано з використанням чистого стеку: HTML5, CSS3 та JavaScript.

## Основні Можливості

* **Персоналізована Генерація Програм:** Створення детальних планів тренувань для всіх 9 основних комбінацій цілей та рівнів підготовки.
* **Багаторівнева Аутентифікація:** Функціонал входу та реєстрації користувачів.
* **Сесійна Незалежність:** Збереження даних користувача (профіль, статистика, програми) у `localStorage` під його унікальним ідентифікатором (логіном).
* **Відстеження Прогресу:** Можливість збереження профілю та оновлення статистики після завершення тренування.
* **Збереження Історії:** Користувач може зберегти будь-яку згенеровану програму до розділу "Мої програми".
* **Налаштування Інтерфейсу:** Функція зміни колірної схеми додатка.

## Технологічний Стек

| Технологія | Призначення |
| :--- | :--- |
| **HTML5** | Основна структура та розмітка сторінки. |
| **CSS3** | Стилізація, адаптивний дизайн, використання CSS-змінних для управління темами. |
| **JavaScript (ES6)** | Вся функціональна логіка: керування станом, аутентифікація, генерація тренувань, взаємодія з `localStorage`. |
